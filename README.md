# ArzResHelper

Хелпер для автоматического отслеживания добычи ресурсов на ферме, шахте и лесопилке, а также для отслеживания заработка с Item Market на проекте **Arizona RP**.


# ✨ Возможности

- 📊 **Отслеживание добычи в реальном времени** — автоматический подсчёт добытых ресурсов на ферме, в шахте (обычной и подземной) и на лесопилке
- 💰 **Подсчёт заработка** — за сессию, сегодня, неделю и всё время с учётом настраиваемых цен для каждого ресурса
- 🎯 **Настройка целей** — индивидуальные дневные цели для каждого ресурса с прогресс-барами и уведомлениями о выполнении
- 🏆 **Система достижений** — 40+ достижений за коллекционирование ресурсов, заработок и выполнение целей (с возможностью сброса прогресса)
- 📈 **Детальная статистика** — просмотр добычи за сегодня, неделю, выбранную дату или всё время
- 📌 **Оверлеи** — компактные перетаскиваемые окна с добычей за сегодня для фермы, шахты и лесопилки
- ⏱️ **Таймер работы** — встроенный таймер в оверлее для отслеживания времени фарма
- 🔊 **Звуковые уведомления** — отдельные звуки для обычных и редких ресурсов, а также при достижении целей
- 🛒 **Item Market** — отслеживание дохода от аренды предметов с историей по дням и неделям
- 📱 **Telegram-уведомления** — уведомления об аренде, ежедневные и еженедельные отчёты со статистикой по всем работам
- 🏆 **Глобальный рейтинг** — таблица лидеров по доходу (день/неделя/всё время) с возможностью участия
- 🌍 **Глобальные цены** — автоматическая загрузка актуальных цен на ресурсы
- 🎨 **Кастомизация интерфейса** — 6 встроенных цветовых тем и редактор собственной темы (выбор цветов через палитру)
- ⌨️ **Биндер клавиш** — назначение горячих клавиш для отправки сообщений и команд с поддержкой переменных и задержек
- 🌙 **Умный сброс** — автоматический сброс дневной статистики в 05:00 МСК с учётом игрового дня





# 📦 Установка

1. Скачай `#ArzResHelper.lua` и помести в папку `moonloader` все автоматически подгрузится при первом запуске игры со скриптом.

# Если что то пошло не так:

1. Скачай `#ArzResHelper.lua` и помести в папку `moonloader`
2. Скачай `moonloader.zip` открой папку `moonloader` и перекинь содержимое этой папки в свой `moonloader`.

# 📝 Важные требования

⚠️ Путь к игре - только латиница! Как собственно для большинства программ на Windows.

Если Аризона установлена в папку, содержащую русские буквы - скрипт будет вылетать с ошибкой. Это связано с тем, что библиотека MoonImGui.dll использует функции Windows API, которые некорректно обрабатывают кириллицу в путях к файлам.

❌ Неправильно (скрипт не запустится):

- D:\Игры\arizona
- C:\Вася\Desktop\arizona
- D:\Софт\Games\arizona

✅ Правильно (всё работает):

- D:\Games\arizona
- C:\GTA\arizona
- D:\Program Files\arizona

Как проверить: откройте папку с игрой и посмотрите на адресную строку проводника — в пути не должно быть ни одной русской буквы.

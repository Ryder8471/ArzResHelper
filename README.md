# ArzResHelper

Хелпер для автоматического отслеживания добычи ресурсов на ферме, шахте и лесопилке на проекте **Arizona RP**.


# ✨ Возможности

- 📊 **Отслеживание добычи в реальном времени** — автоматический подсчёт добытых ресурсов
- 💰 **Подсчёт заработка** — за сессию, сегодня, неделю и всё время
- 🎯 **Настройка целей** — индивидуальные цели для каждого ресурса с прогресс-барами
- 🔊 **Звуковые уведомления** — отдельные звуки для обычных и редких ресурсов, а также при достижении целей
- 📌 **Оверлеи** — компактные перетаскиваемые окна с добычей за сегодня
- ⏱️ **Таймер работы** — встроенный таймер для отслеживания времени фарма
- ⌨️ **Биндер клавиш** — назначение горячих клавиш для отправки сообщений/команд


# 📦 Установка

1. Скачай `#ArzResHelper.lua` и помести в папку `moonloader`
2. Скачай `moonloader.zip` и распакуй в корневую папку Аризоны (создаст папку `ResHelper` с файлами и добавит звуки в `resource/farm/`)
3. Или: Скачай `moonloader.zip` открой папку `moonloader` и перекинь содержимое этой папки в свой `moonloader`.

# 📝 Важные требования

⚠️ Путь к игре - только латиница! Как собственно для большинства программ на Windows.

Если Аризона установлена в папку, содержащую русские буквы - скрипт будет вылетать с ошибкой. Это связано с тем, что библиотека MoonImGui.dll использует функции Windows API, которые некорректно обрабатывают кириллицу в путях к файлам.

❌ Неправильно (скрипт не запустится):

- D:\Игры\arizona
- C:\Вася\Desktop\arizona
- D:\Софт\Games\arizona

✅ Правильно (всё работает):

- D:\Games\arizona
- C:\GTA\arizona
- D:\Program Files\arizona

Как проверить: откройте папку с игрой и посмотрите на адресную строку проводника — в пути не должно быть ни одной русской буквы.

# ArzResHelper

Хелпер для автоматического отслеживания добычи ресурсов на ферме, шахте и лесопилке, учёта заработка с Item Market, осколков тайников и пейдеев на проекте **Arizona RP**. Включает глобальный рейтинг, систему достижений, Telegram-отчёты и привязку статистики к серверу.


# ✨ Возможности

- 📊 **Отслеживание добычи в реальном времени** — автоматический подсчёт добытых ресурсов на ферме, в шахте (обычной и подземной) и на лесопилке
- 💰 **Подсчёт заработка** — за сессию, сегодня, неделю и всё время с учётом настраиваемых цен для каждого ресурса
- 🎯 **Настройка целей** — индивидуальные дневные цели для каждого ресурса с прогресс-барами и уведомлениями о выполнении
- 🏆 **Система достижений** — 40+ достижений за коллекционирование ресурсов, заработок и выполнение целей (Steam-подобные уведомления)
- 📈 **Детальная статистика** — просмотр добычи за сегодня, неделю, выбранную дату или всё время
- 📌 **Оверлеи** — компактные перетаскиваемые окна с добычей за сегодня для фермы, шахты и лесопилки (2 стиля оформления)
- ⏱️ **Таймер работы** — встроенный таймер в оверлее для отслеживания времени фарма
- ⛏️ **Таймер спавна** — отображение времени до спавна руды и големов в подземной шахте
- 🔊 **Звуковые уведомления** — отдельные настраиваемые звуки для обычных, редких и очень редких ресурсов
- 🛒 **Сторонний доход** — отслеживание дохода от аренды предметов (Item Market), осколков тайников и пейдеев
- 💳 **Учёт пейдеев** — зарплата, депозит, аксессуары/предметы и AZ с детализацией по источникам
- 📱 **Telegram-уведомления** — уведомления об аренде и осколках, ежедневные и еженедельные отчёты с обходом блокировок
- 🏅 **Глобальный рейтинг** — таблица лидеров по доходу с привязкой к серверу и иконками серверов Arizona RP
- 🌍 **Глобальные цены** — автоматическая загрузка актуальных цен на ресурсы из Google Таблицы
- 🖥️ **Новый интерфейс** — современный дизайн 1020x620, перетаскивание окна, категории в меню, Steam-уведомления
- 🎨 **Кастомизация** — 6 цветовых тем + редактор собственной темы с палитрой цветов
- ⌨️ **Биндер клавиш** — назначение горячих клавиш для отправки сообщений и команд с переменными и задержками
- 🌐 **Привязка к серверу** — отдельная статистика для каждого сервера Arizona RP, авто-миграция старых данных
- 🌙 **Умный сброс** — автоматический сброс дневной статистики в 05:00 МСК с учётом игрового дня





# 📦 Установка

1. Скачай `#ArzResHelper.lua` и помести в папку `moonloader` все автоматически подгрузится при первом запуске игры со скриптом.

# Если что то пошло не так:

1. Скачай `#ArzResHelper.lua` и помести в папку `moonloader`
2. Скачай `moonloader.zip` открой папку `moonloader` и перекинь содержимое этой папки в свой `moonloader`.

# 📝 Важные требования

⚠️ Путь к игре - только латиница! Как собственно для большинства программ на Windows.

Если Аризона установлена в папку, содержащую русские буквы - скрипт будет вылетать с ошибкой. Это связано с тем, что библиотека MoonImGui.dll использует функции Windows API, которые некорректно обрабатывают кириллицу в путях к файлам.

❌ Неправильно (скрипт не запустится):

- D:\Игры\arizona
- C:\Вася\Desktop\arizona
- D:\Софт\Games\arizona

✅ Правильно (всё работает):

- D:\Games\arizona
- C:\GTA\arizona
- D:\Program Files\arizona

Как проверить: откройте папку с игрой и посмотрите на адресную строку проводника — в пути не должно быть ни одной русской буквы.
